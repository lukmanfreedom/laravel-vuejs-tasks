# Laravel-VueJS-Tasks App
===

## Built using:
```
Laravel 5.7.6
VueJS 2
Bootstrap 4
```

## Project setup
```
composer install
```

## Migrating Database
```
php artisan migrate
```

### Compiles and hot-reloads for development
```
npm run watch
```

### Compiles and minifies for production
```
npm run dev
```
